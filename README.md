# 🌬️ Respiro - Guide d'installation rapide

Bienvenue sur **Respiro**, votre outil de respiration contrôlée et de cohérence cardiaque, conçu pour fonctionner partout, tout le temps, même sans aucune connexion Internet (Mode Avion).

Pour profiter pleinement de l'application sur votre appareil et l'installer comme une application native, suivez le guide ci-dessous.

---

## 📱 Installation sur Smartphone et Tablette

### 🍏 Sur iPhone / iPad (Safari)
1. Ouvrez le lien de l'application dans le navigateur **Safari**.
2. Attendez **10 secondes** sur la page sans rien toucher (le temps que l'application télécharge les sons en arrière-plan).
3. Appuyez sur le bouton **Partager** 📤 (le carré avec une flèche vers le haut au bas de l'écran).
4. Faites défiler le menu vers le bas et choisissez **Sur l'écran d'accueil** ➕.
5. Donnez-lui le nom "Respiro" et validez.
6. **Test ultime :** Fermez Safari, lancez l'application depuis votre écran d'accueil, passez en **Mode Avion** et respirez !

### 🤖 Sur Android (Chrome)
1. Ouvrez le lien de l'application dans le navigateur **Google Chrome**.
2. Attendez **10 secondes** sur la page.
3. Appuyez sur les **3 petits points** 💬 en haut à droite.
4. Choisissez **Ajouter à l'écran d'accueil** ou **Installer l'application**.
5. **Test ultime :** Fermez Chrome, ouvrez l'application depuis votre écran d'accueil, passez en **Mode Avion** et profitez !

---

## 💻 Installation sur Ordinateur (PC / Mac)
1. Ouvrez le lien dans **Chrome**, **Edge** ou **Brave**.
2. Dans la barre d'adresse tout à droite, cliquez sur la petite icône d'ordinateur avec une flèche (ou l'icône **"+"**).
3. Cliquez sur **Installer**. L'application aura désormais sa propre fenêtre indépendante.

---
⚠️ *Note : Pour les futures mises à jour, il vous suffira d'ouvrir l'application une fois en étant connecté à Internet pour qu'elle se mette à jour toute seule en tâche de fond.*
